# Infinity Pulse ⚡

A futuristic 3D Counter App built using HTML, CSS, JavaScript, and Three.js.

Infinity Pulse transforms a simple counter into an immersive visual experience with animated particles, orbital rings, floating 3D objects, and smooth UI interactions.

---

## 🚀 Features

- Increment & Decrement Counter
- Reset Functionality
- Interactive 3D Background
- Floating Particle Effects
- Animated Orbital Rings
- Smooth Counter Animations
- Responsive Modern UI
- Keyboard Support
- Glassmorphism Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Three.js
- Font Awesome

---

## ⌨️ Keyboard Controls

| Key | Action |
|------|---------|
| ↑ / → | Increment |
| ↓ / ← | Decrement |
| 0 / ESC | Reset |
